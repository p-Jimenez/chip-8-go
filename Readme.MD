# Chip-8 Go Emulator

This is a Chip-8 emulator written in Go. It is a work in progress.

## Usage

```
go run . <path to rom>
```

## Resources

- [Cowgod's Chip-8 Technical Reference](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM)
